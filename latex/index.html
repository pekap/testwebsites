For any damaged network with power vector $P$ there exists pre-contingency network that has the same structure except tripped contingecy lines with power vector $\tilde P$.
If there are several contingency lines in set $C$ then we can easily represent one power vector via another:

(I+\delta _{C} B ^{-1} )\tilde P=P

Where $\delta _{C} $ is a minor of matrix $B$ that correspons to contingency set $C$. Since such minors have only $|C|$ non-zero rows we can represent $\delta _{C} B^{-1} $ in the following way:

\delta _{C} B ^{-1} = U _{C} V _{C} ^{T}

Where $U _{C}$ and $V _{C}$ (both are of size $N_{buses} \times |C|$ are the parts of matrices $U,V$ (square matrices of size $ N_{buses} \times N_{buses} $). Then the relationship between $P$ and $\tilde P$ can be given in the following way:


\tilde P = P - U_C [I + V_C^T U_C]^{-1} V_C^T P

Lets do some denotions:

V_C^T U_C = A_C = Diag(A_C) + NonDiag(A_C) = d_C + n_C
V_C^T U_C = d_C + n_C

Then line flows can be represented as following:


F = - Y U^{T} B^{-1} P
N_L \times 1 = [N_L \times N_L] [N_L \times N_B] [N_B \times N_B] [N_B \times 1]

Which means that new line flows (after contingecy) are equal to:


\tilde F = F + Y U^{T} B^{-1} U_C [I+V_C^T U_C] ^{-1} V_C^T P

Lets denote $\Gamma_C = Y U^{T} B^{-1} U_C$ and $q_C=V_C^T P$ then:


\tilde F = F + \Gamma_C [I+A_C] ^{-1} q_C

Which means that in order to assure safety of the contigency we should prove:

-F_{Lim} - F < \Gamma_C [I+A_C] ^{-1} q_C < F_{Lim} - F

Having two diagonal matrices that represents Limitations $L_{up}=Diag(F_{Lim}-F)$ and $L_{bot}=Diag(-F_{Lim}-F)$ we get:


L_{up}^{-1} \Gamma_C [I+A_C] ^{-1} q_C < \boldsymbol{1}
L_{bot}^{-1} \Gamma_C [I+A_C] ^{-1} q_C < \boldsymbol{1}

Lets exploit diagonal and nondiagonal representation of the matrix $A_C$

[I+d_C+n_C]^{-1} = (1+d_C)^{-1} - (1+d_C)^{-1} n ((1+d_C)^{-1} n +1)^{-1}

Denoting $\Delta_C=(1+d_C)^{-1} n$ we get

[I+d_C+n_C]^{-1} = (1+d_C)^{-1} - \Delta_C (\Delta_C+1)^{-1}

So finally we get:

L^{-1} \Gamma_C (1+d_C)^{-1} q_C - L^{-1} \Gamma_C \Delta_C (\Delta_C+1)^{-1} q_C < \boldsymbol 1
